# oros_appDev
online_grocery_ordering_system

---

## Tech stack
- frontend: React
- backend: Spring Boot

## Backend project
Located in `app/`.

## Prerequisites
- Java 17+ installed
- Maven installed or use the included wrapper
- Optional: Postman or curl for API testing

## Run
```bash
cd app
./mvnw clean compile
./mvnw spring-boot:run
```
